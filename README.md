## Acerca de

Este repo tiene las siguientes tres funciones:

1. **Bitácora y práctica de código** — registrar cada sesión de estudio (`bitacora/`) y ejercicios guiados (`notebooks/`) mientras avanzo en mi ruta de aprendizaje.
2. **Práctica del control de versiones** — a falta de conocimiento en ciencia de datos y, por tanto, de cómo diseñar un proyecto aplicado de ciencia de datos, practicar el control de versiones con Git/GitHub desde el primer día.
3. **Índice de proyectos** — una vez que tenga la experiencia y empiece a construir proyectos end-to-end, será el banco de mis proyectos de ciencia de datos, cada uno en su propio repositorio; acá solo se encontrarán los links a dichos repos.

### Cómo está organizado

| Carpeta      | Contenido                                                                                 |
| ------------ | ----------------------------------------------------------------------------------------- |
| `bitacora/`  | Una entrada por día de estudio: qué estudié, qué me costó, qué queda pendiente.           |
| `notebooks/` | Ejercicios guiados de práctica, organizados por fase y tema. No son proyectos end-to-end. |
| `data/`      | Datasets pequeños usados en los notebooks de práctica.                                    |
| `src/`       | Código reutilizable (funciones, clases) que se repite entre notebooks.                    |
| `reports/`   | Salidas puntuales de algún ejercicio que valga la pena mostrar aparte del notebook.       |
